## How to use

1. Open the executed notebook on GitHub (or in Colab) on one half of your screen.
2. Open the matching script on the other half.
3. Start your screen recorder.
4. Read the **Intro**, then go cell by cell — scroll to each cell as you read its paragraph.
5. Pause on code-cell outputs so viewers can see the printed values or plots while you finish that cell's narration.
6. Close with the **Outro**.

## Tone notes

- Scripts are written in a natural, conversational first-person tone ("I'm Nitish, in this video I'm walking through…").
- Every code cell's expected output is described, so you can point to it on-screen as you talk.
- HTML-styled markdown cells in the notebooks are summarized at the concept level, not read verbatim.
- Feel free to paraphrase — the scripts are a guide, not a teleprompter.
